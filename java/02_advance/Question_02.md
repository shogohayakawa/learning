# Question2

## 開発要件
1. あるパン屋の一日の売上の合計（税込み）、販売個数の最も多い商品番号、販売個数の最も少ない商品番号を出力するプログラムを作成してください。
2. 商品は10種類あり、それぞれの税抜単価は以下の通りとしてください。商品番号は```1〜10```の整数とします。
```bash
# 商品単価（税抜）
1.  100
2.  120
3.  150
4.  250
5.   80
6.  120
7.  100
8.  180
9.   50
10. 300
```
3. 入力は```コマンドライン引数```から、以下の形式で与えられるように作成してください。
```bash
# 販売した商品番号 販売個数 販売した商品番号 販売個数...
1 10 2 3 5 1 7 5 10 1
```
4. 消費税率は```10%```としてください。
5. 販売個数の最も多い商品と販売個数の最も少ない商品について、販売個数が同数の商品が存在する場合、それら全ての商品番号を出力してください。
6. プログラムの実行結果は、以下と同様になるよう作成してください。

## コンソール出力結果
```bash
# 売上の合計
# 販売個数の最も多い商品番号
# 販売個数の最も少ない商品番号

# コマンドライン引数の入力例
# 1 10 2 3 5 1 7 5 10 1
2464
1
5 10
```
